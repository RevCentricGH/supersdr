# Bundled assets

Drop your agency's brand assets here. Paths in `config.json` are resolved relative to the
config file, so reference them like `assets/logo.svg` (not an absolute path). Asset paths may
not point outside this skill folder.

- `logo.svg` - your agency logo (shown on the title slide). Replace the placeholder.
- `clients/` - one image per client logo listed under `proof.client_logos`.

PNG or SVG both work. Keep logos reasonably small so the rendered deck stays light.
